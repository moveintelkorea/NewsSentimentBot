# 주식 뉴스 감정분석 텔레그램 봇

## 콘텐츠 개요

**채널:** 빅파이썬  
**주제:** 네이버 검색 API로 주식 뉴스를 수집해 AI 감정분석 후, 텔레그램으로 하루 2번 자동 발송하는 봇  
**난이도:** 중급  
**예상 영상 길이:** 30~40분

---

## 프로젝트 목표

- 네이버 Open API(뉴스 검색)로 주요 종목별 뉴스 수집
- OpenAI API로 뉴스 감정(긍정/부정/중립)을 분석
- 매일 **오후 9시** — 긍정 뉴스 Top 10 / 부정 뉴스 Top 10 종목 발송
- 다음날 **오후 3시 10분** — 어제 Top 10 종목들의 당일 주가 변화율 발송

---

## 알림 스케줄 (하루 2회)

```
21:00  📰 뉴스 감정 분석 리포트
       ├── 긍정 뉴스 Top 10 종목
       └── 부정 뉴스 Top 10 종목

15:10  📈 어제 예측 결과 확인
       ├── 어제 긍정 Top 10 → 오늘 주가 변화율
       └── 어제 부정 Top 10 → 오늘 주가 변화율
```

## 기술 스택

| 항목 | 기술 |
|------|------|
| 언어 | Python 3.11+ |
| 뉴스 수집 | 네이버 Open API (뉴스 검색) |
| 감정 분석 | OpenAI API (`gpt-4o-mini`) |
| 주가 데이터 | `yfinance` (KRX 종목 코드 기반) |
| 텔레그램 봇 | `python-telegram-bot` |
| 스케줄러 | `APScheduler` |
| 환경변수 관리 | `python-dotenv` |
| 데이터 저장 | JSON 파일 (전날 Top 10 종목 보관) |

## 핵심 플로우

### 오후 9시 — 뉴스 감정 분석 리포트

```
1. 감시 종목 리스트에서 각 종목명으로 네이버 뉴스 검색 (최신 5개)
2. OpenAI API로 각 뉴스 감정 점수 산출 (-1.0 ~ +1.0)
3. 종목별 평균 감정 점수 집계
4. 긍정 점수 상위 10개 / 부정 점수 하위 10개 선정
5. 텔레그램으로 리포트 발송 + 결과를 JSON에 저장
```

### 오후 3시 10분 — 주가 변화율 확인

```
1. 어제 저장된 긍정/부정 Top 10 종목 JSON 로드
2. yfinance로 각 종목의 당일 변화율 조회
3. 텔레그램으로 결과 발송 (예측 vs 실제 비교)
```

## 개발 단계

### Phase 1 — 뉴스 수집 (1일차)
- [ ] 네이버 Open API 뉴스 검색 구현 (`crawler.py`)
- [ ] 종목명 키워드로 최신 뉴스 N개 수집
- [ ] 응답 파싱 및 HTML 태그 제거

### Phase 2 — 감정 분석 (2일차)
- [ ] OpenAI API 연동 (`analyzer.py`)
- [ ] 뉴스 → 감정 점수 프롬프트 설계
- [ ] 종목별 평균 점수 집계 및 Top 10 선정

### Phase 3 — 주가 조회 (2일차)
- [ ] yfinance로 당일 변화율 조회 (`stock.py`)
- [ ] 어제 Top 10과 오늘 주가 비교
- [ ] 예측 적중률 계산

### Phase 4 — 텔레그램 봇 (3일차)
- [ ] 텔레그램 봇 토큰 발급 및 기본 연동 (`bot.py`)
- [ ] 오후 9시 감정 리포트 메시지 포맷
- [ ] 오후 3시 10분 주가 변화율 메시지 포맷

### Phase 5 — 스케줄러 및 통합 (4일차)
- [ ] APScheduler 21:00 / 15:10 설정 (`scheduler.py`)
- [ ] Top 10 결과 JSON 저장/로드 (`storage.py`)
- [ ] 전체 파이프라인 통합 테스트
- [ ] 예외 처리 및 에러 텔레그램 알림

## 영상 구성 (시나리오)

1. **인트로** (2분) — 완성본 데모 (텔레그램 알림 2종 실제 시연)
2. **API 준비** (5분) — 네이버 Open API 키 발급, 텔레그램 봇 생성
3. **뉴스 크롤러** (7분) — 네이버 뉴스 검색 API 연동
4. **OpenAI 감정 분석** (8분) — 프롬프트 설계 + Top 10 선정 로직
5. **주가 변화율 조회** (5분) — yfinance로 다음날 결과 확인
6. **텔레그램 발송** (5분) — 메시지 포맷 및 전송
7. **스케줄러 자동화** (4분) — 하루 2회 자동 실행
8. **마무리** (4분) — 전체 흐름 정리 및 개선 아이디어
